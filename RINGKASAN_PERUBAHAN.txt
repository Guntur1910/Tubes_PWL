Proyek: Tubes_PWL

================================================================================
1. DATABASE MIGRATIONS
================================================================================

- Tambahan kolom pada tabel ticket_type:
  - event_id (foreign key ke events)
  - name
  - description
  - price
  - quota
  - sold
  - sales_start
  - sales_end

- Pembaruan tabel transactions:
  - ditambahkan ticket_type_id (foreign key ke ticket_type)
  - status enum disesuaikan menjadi ['pending','paid','failed']
  - data transaksi pending digunakan sebagai keranjang

- Tabel tickets:
  - ditambahkan migration baru untuk tabel tickets dengan kolom:
    transaction_id, ticket_code, qr_code_path, status, used_at, timestamps
  - jika tabel sudah ada, migration ini menjaga agar tidak mengulang pembuatan tabel

- Tabel waiting_lists:
  - ditambahkan migration baru untuk waiting list dengan kolom:
    user_id, ticket_type_id, quantity, status, notified_at, expires_at, timestamps
  - migration diperbaiki agar aman jika tabel waiting_lists sudah ada

3. CONTROLLERS
================================================================================

- app/Http/Controllers/EventController.php
  - show() menampilkan event dengan ticketTypes
  - buyTicket() memproses add to cart dengan status pending
  - updateQuantity() endpoint AJAX untuk ubah quantity keranjang
  - deleteFromCart() endpoint untuk hapus item keranjang

- app/Http/Controllers/User/HomeController.php
  - checkout() menampilkan cart berdasarkan status pending
  - payCheckout() memproses pembayaran QRIS, mengubah status ke paid,
    kemudian generate ticket dan kirim email

- app/Http/Controllers/TicketController.php
  - __construct() memakai middleware auth
  - generateTickets() membuat tiket individual dan QR code
  - myTickets() menampilkan tiket user
  - scanPage() menampilkan halaman scan ticket admin
  - validateTicket() memvalidasi kode tiket dan menandai sebagai used
  - joinWaitingList() menambahkan user ke waiting list jika stok habis
  - sendTicketEmail() mengirim email dengan lampiran QR code

- app/Http/Controllers/Controller.php
  - diubah agar extends Illuminate\Routing\Controller dan memakai trait standar
    AuthorizesRequests, DispatchesJobs, ValidatesRequests
